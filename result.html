I have a lot of experience in this area. Memory palaces can be useful for things that don't really matter. names and phone numbers are not important. Important information, unless really simple, should not be used in a memory palace. I want a good memory, so I can pass an exam. it is a complicated exam that does not rely on simple fact recall. it tests for my understanding. the process of making a memory palace for a subject I'm studying and the process for developing an understanding of a subject are not easily linked. Therefore, if it is important information, for example a subject you are studying, it is more effective to do it the traditional way. The reason a memory palace does not aid developing understanding is because the information is coded into imagery, which the brain's subconscious mostly won't interpret. I'm sure there will be a lot of people who disagree with me, perhaps you have other experiences. please share.spam
